# MK-FORMPD88 - PEDANG88 Landing Page

Landing page resmi yang dioptimalkan untuk performa tinggi menggunakan framework **AMP (Accelerated Mobile Pages)**. Proyek ini dirancang untuk kecepatan akses mobile dan optimasi mesin pencari (SEO).

## 🚀 Fitur Utama
- **AMP Powered**: Loading super cepat dan ramah perangkat mobile.
- **SEO Optimized**: Mengejar kata kunci utama *PEDANG88* dan *temukanaja formpd88*.
- **Analytics Ready**: Terintegrasi dengan Google Analytics 4 (GA4) dan Meta Pixel (1579939523185127).
- **Clean Metadata**: Optimasi Open Graph (OG) untuk pratinjau media sosial yang menarik.

## 🛠️ Teknologi yang Digunakan
- HTML5 & AMP Project
- CSS3 (AMP-Custom)
- JSON-LD (Structured Data)

## 📂 Struktur Proyek
- `index.html`: File utama landing page.
- `LICENSE.md`: Lisensi proyek (MIT).
- `README.md`: Dokumentasi proyek.

## 🌐 Deployment
Situs ini dideploy menggunakan **GitHub Pages**. Setiap perubahan pada branch `main` akan secara otomatis memperbarui situs live.
